# ApplePractice

A premium practice landing page built with Next.js, Tailwind CSS, Framer Motion, lucide-react icons, and a free no-key Open-Meteo API panel.

## Run locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Stack

- Next.js
- Tailwind CSS
- Framer Motion
- lucide-react
- Open-Meteo public weather API

This is an original practice project inspired by polished product-site pacing, not an Apple clone.
